Python is simple, easy to use, interpreted programming language, offering much more structure and support for large programs than shell scripts or batch files can offer. Being a very-high-level language, it has high-level data types built in, such as flexible arrays and dictionaries. Because of its more general data types Python is applicable to a much larger problem domain.

     As Python is an interpreted language, which can save you considerable time during program development because no compilation and linking is necessary. Python allows you to split your program into modules that can be reused in other Python programs. It comes with a large collection of standard modules that you can use as the basis of your programs or as examples to start learning to program in Python. Some of these modules provide things like file I/O, system calls, sockets, and even interfaces to graphical user interface toolkits like Tk. Python also has Web Development Frameworks like Django, Flask and Pyramid. It can also be used for automation with tools like Robot Framework, Selenium and Ansible.

List - A Data Structure

List is Data structure in python, like Array. But Unlike String and Array, which are 'Immutable' list is 'Mutable'. It store items separated by comma(,), It might contain items of same type or different types. 
>>> team = [3, 9, 11, 19, 37]
>>> team
[3, 9, 11, 19, 37]
>>> team[1]
9
>>> team[-1]
37
>>> team[2] = 91
>>> team
[ 3, 9, 91, 19, 37]
>>> team.append('b')
>>> team
[ 3, 9, 91, 19, 37, 'b']
>>> team.pop()
'b'
