Now we will focus on important features, break, continue, pass Statements.

break Statement

     break can used to break the loops like for, while. It takes control from loops to end of the loop
Example: 
>>> for i in range(10):
...     if i%2 == 0:
...         break
...     print i,
...

continue Statement

     continue is also break the loops, but it takes control from loops to start of loop.
Example: 
>>> for i in range(10):
...     if i%2 != 0:
...         continue
...     print i,
...
Output Will be : 0 2 4 6 8

pass Statement

     pass can be used when statement require but without action, like we create base class that provides base for inheriting class.
Example: 
>>> def foo():
...     pass
...
This will print nothing.
